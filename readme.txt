A dynamic navigation widget for those using WordPress as a web site as opposed to a basic blog.

== Description ==

CCS-NAVIGATION is a dynamic navigation widget that is placed in the sidebar of your theme.  This
widget builds a navigation structure for the current page (links to sub-pages) and each
of the ancestor level pages.  The easiet way to explain this is to simply try it out or
check out the products section of our website to see it in action:
   http://www.creativecloudsolutions.com/products/



== Installation ==

1. Upload the entire 'ccs-navigation' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the 'CCS Navigation' widget in your side bar
4. That's it!  It is so simple, there are no settings to mess with.
